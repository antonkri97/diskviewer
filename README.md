# Одностраничное приложение для просмотра директорий Яндекс.Диска

## Используемые технологии

* React
* Redux
* React-Router 3.0
* React-Router-Redux
* Redux-Thunk
* React-Bootstrap
* Styled-Components

## Запуск приложения

* git clone https://github.com/antonkri97/diskviewer
* cd diskviewer
* yarn / npm install
* yarn / npm start

__Важно! Проект обязательно должен быть запущен на 3000 порту!__
