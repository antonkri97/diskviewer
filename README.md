# Одностраничное приложение для просмотра директорий Яндекса.Диска.

## Используемые технологии

* React
* Redux
* React-Router 3.0
* React-Router-Redux
* Redux-thunk
* React-Bootstrap
* Styled-Components

## Запуск приложения

* git clone https://github.com/antonkri97/diskviewer
* yarn / npm install
* yarn / npm start

__Проверьте, что проект запущен на 3000 порту!__
